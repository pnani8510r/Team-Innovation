🌟Team Innovative Presents: LinguaQuest
  🔹 Main Features
    🪙 Coin System

     Users earn coins by solving problems and explaining doubts.

     Coins can be redeemed into national currency.

    ❓ Doubt Solving Forum

        Users can raise doubts.

        Doubts can be answered by students or teachers.

        Creates a comfortable learning environment.

    🎥 Live Sessions

        Doubt solvers can create a video call session.

        Enables real-time explanation.

    ⚔️ Contest Rooms

        Friends can compete with each other.

        Winners earn coins.

    🎯 Daily Problems (Mood-Based)

        5 questions selected based on user’s mood.

        Easy questions if user feels low to increase interest.

    📝 Notes Section

        Users can write what they learned for every question.

    🤖 Chatbot

        Helps to solve problems instantly.

    🏅 Badges & Streaks

        Earn badges by maintaining daily streaks.

        Helps users stand out.


                  🌟 Standout Features

       -> Coin System with Currency Conversion

       -> Live Doubt Solving via Video Call

       -> Contest Rooms for Competitive Learning# Team-Innovation
